Integration notes

How to use this dataset in your AI / RAG / compliance pipelines.

  • Read JSON directly from datasets/ai-act/v1/articles/XX.json in your pipeline.
  • Use id (for example AI_ACT_ART_01) as a stable reference in prompts and citations.
  • Use keywords for routing or pre-filtering before vector search.
  • version and last_updated let you pin a specific dataset version for audits.
