Proof of dataset

For each dataset version, DatasetCenter produces a cryptographic proof:

  • SHA-256 hash of JSON/CSV files
  • Timestamped, signed proof (.tproof.json)
  • Public verification endpoint via TimeProofs
  • Audit trail suitable for AI Act documentation
